opne-loop transfer function(개방루프 전달함수)는 $\frac{Y(s)}{R(s)}$ = $\frac{K}{s+50}$ 이다  
$R(s)$ = $\frac{1}{s}$ 이기에 $Y(s)$ = $\frac{K}{s(s+50)}$ 이다  
부분분수로 나타내면 $Y(s)$ = $\frac{K}{50}(\frac{1}{s}-\frac{1}{s+50})$ 이고  
윗부분을 라플라스 변환을 하면 $y(t)=\frac{K}{50}(1-e^{-50t})$ 이다
